Acerca del proyecto

Este prototipo ha sido creado como parte de uno mayor que en un futuro permitirá crear, editar y mantener una biblioteca de hojas de personaje para juegos de rol de sobremesa.

Es un proyecto encargado a los alumnos del primer curso del ciclo formativo de Desarrollo de Aplicaciones Web del IEFPS Elorrieta - Erreka Mari, para la clase de Entornos de Desarrollo.

El proyecto no incluye como objetivo la realización del código propiamente dicho, pero en un futuro espero terminarlo como un proyecto personal.

Herramientas utilizadas para el prototipo de interface

Bootstrap

La base del prototipo es el famoso framework Bootstrap de twitter, que me permite utilizar sus elementos para crear el interfaz rápidamente y sin preocuparme demasiado del código, especialmente del CSS y el javascript.

Bootstrap hace sencillo crear páginas web y aplicaciones que funcionen en todos los navegadores y dispositivos usando responsive CSS.

Con Bootstrap vienen incluidos los Glyphicons que también he utilizado.

Git

Se ha sometido a todos los elementos de configuración de software a un control de cambios, utilizando para ello la herramienta git. No era un objetivo de la actividad, pero los CVS sí son parte de la asignatura y tampoco emprendería ningún proyecto que me llevara más de diez minutos sin llevar un control.

La mayor ventaja de git frente a otros CVS es la posibilidad de creación de ramas, que permiten trabajar con diferentes partes de un proyecto simultáneamente y experimentar sin miedo a perder el trabajo realizado en las otras ramas, especialmente la principal o master.

Otra ventaja es que es distribuido, todos los usuarios tienen una copia del repositorio completo en local. A pesar de que el único usuario que trabaja en el proyecto soy yo, esto me ha permitido poder trabajar en el ordenador de casa y en las diferentes particiones del ordenador del centro sin problema y sin miedo a perder los datos en caso de corrupción o borrado accidental.

Tampoco es necesaria la existencia de un servidor central aunque…

GitHub

… GitHub me ha permitido tener uno igualmente. GitHub es una comunidad que permite tener múltiples repositorios gratuitos de código abierto, accesibles a través de Internet. Esto facilitaría mucho la tarea de trabajar en equipo si el proyecto no fuera individual, pero también ofrece herramientas muy interesantes de visualización de revisiones, ramas, ficheros… incluso tiene su propio issue tracker, en el que los hipotéticos usuarios o contribuyentes al código podrían señalar problemas y contribuir código.

Es interesante (aunque algo avergonzante para un novato como yo) poder ver toda la evolución que ha ido teniendo el proyecto en todas sus fases.

Las comunidades de colaboración en código abierto son también objetivo de estudio en la asignatura y por eso he publicado este proyecto como tal.
